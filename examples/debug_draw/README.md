# Debug Draw

In this example application, we demonstrate how to draw oriented 3D debug bounding boxes around all components in the current scene. You can run this example as follows.

```console
python run.py
```

You should see a game window appear, with randomly colored debug boxes drawn around each component. Note that Unreal's debug drawing functionality is disabled in Shipping builds, and therefore this example is only expected to work in Development builds and in play-in-editor sessions in the Unreal Editor.
