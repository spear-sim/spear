# Editor Script Interoperability

In this example application, we demonstrate how to evaluate Python expressions in the Unreal Editor using SPEAR and send various data types back and forth between SPEAR Python and Editor Python. You can run this example as follows.

```console
python run.py
```

You should see the Unreal Editor open, with debug output printed to the console showing each type being evaluated in the editor and returned to the SPEAR program.
